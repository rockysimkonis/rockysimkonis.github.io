Enumerable Map Method

  • Rocky S.
  • Feb 1, 2016
  • Sharable links

The map method is on of the most powerful and useful enumerable methods available. The beauty in this method is that whatever enumerable it starts with, map always returns an array. Plus the returned array is always the same size as the original enumerable.

Here is an example
languages = %w {Ruby Java C C++ Python}
>> ["Ruby","Java","C","C++","Python"]
languages.map {|language| language.upcase }
>> ["RUBY","JAVA","C","C++","PYTHON"]
